We've added two new sections to walshit.com: **Updates** and **Guides**.

**Updates** is where planned maintenance and other service notes will be
posted, in reverse-chronological order, with an RSS feed if you'd rather
subscribe than check back. Active incidents still go out over ntfy/Discord
and show up on the Framerr status page first — this page is for the
planned, scheduled stuff and after-the-fact notes, not real-time alerts.

**Guides** collects the small number of evergreen how-tos worth keeping
around, starting with getting set up and requesting media. Nothing here
needs a search box or pagination; if the list gets long, we'll take that
as a sign to prune it.

Both sections are plain, static pages built the same way as the rest of
the site — no accounts, no comments, no tracking.
